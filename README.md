# 📊 Investment Journal (Qt + C++ + SQLite)

Простое desktop-приложение для учета личных инвестиций, написанное на **Qt 6 (C++)** с использованием **SQLite**.

---

## 🚀 Описание проекта

Investment Journal — это приложение, которое позволяет вести журнал своих инвестиций:

- добавлять сделки (актив, сумма, цена, дата)
- просматривать список инвестиций в таблице
- хранить данные локально в SQLite базе

Проект создан как **pet-project для изучения Qt Widgets и работы с SQL в Qt**.

---

## 🧰 Технологии

- C++17
- Qt 6.8.2
- Qt Widgets
- Qt SQL (QSQLITE)
- SQLite (локальная база данных)

---

## 📸 Функционал (MVP)

- ➕ Добавление инвестиции
- 📋 Отображение всех записей в таблице
- 💾 Хранение данных в локальной базе SQLite
- 🔄 Автообновление таблицы после добавления

---

## 🗂️ Структура проекта
```
InvestmentJournal/
    ├── main.cpp
    ├── mainwindow.h / mainwindow.cpp
    ├── mainwindow.ui
    ├── database.h / database.cpp
    ├── CMakeLists.txt (или .pro файл)

```
---

## 🗄️ Структура базы данных

Таблица investments:

- id (INT, PK)
- asset (TEXT)
- amount (REAL)
- price (REAL)
- date (TEXT)
